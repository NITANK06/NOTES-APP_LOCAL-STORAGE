 **Notes App with LocalStorage**

 **Objective**
A simple Notes App built using HTML, CSS, and JavaScript that stores notes in browser localStorage.

 Features
  Input validation
  Add notes
  Delete notes
  Responsive UI
  Persistent storage

 **Technologies stack**
  JavaScript (ES6)
  CSS3
  LocalStorage API
  HTML5

 **How to Run**
1. Clone the repo
2. Open index.html in browser
